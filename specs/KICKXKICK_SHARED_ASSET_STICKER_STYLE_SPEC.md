# Kick×Kick Shared Asset / Sticker Style仕様

更新日: 2026-06-29

詳細な画面操作と上限は `KICKXKICK_MVP_STICKER_COLLECTION_SPEC.md` を正本とする。

## 共有画像資産

- 原本写真は変更しない。
- 背景削除済みの高画質透過PNGをCollection、Detail、TOP5、今日履いた靴、Stickerで共有する。
- Sticker Board表示では原本から透明余白を切り詰めて生成した最大640pxの透過プレビューをキャッシュ利用する。
- 再編集と将来の単体PNG出力では原本透過PNGを使用する。

## 標準Sticker Style

- 靴切り抜き → 白い内フチ → 太いオレンジ外フチ → 任意の柔らかいShadow。
- テキストはオレンジ文字 → 白フチ → オレンジ外フチ。
- テキストは靴ステッカーのフチ内へ配置する。
- テキストのサイズ・位置・色・フチ色・Shadowはステッカー単位で保持する。

## 整合性

- 旧仕様の「文字を靴画像の外側へ配置」は廃止する。
- 旧上限20枚／100枚は廃止し、Free 10枚／Premium 30枚へ統一する。
- 画面表示用プレビューは表示最適化であり、原本の画質を変更しない。
