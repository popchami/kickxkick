# Kick×Kick MVP Sticker / Collection 正式仕様

更新日: 2026-06-29

本書をMVPにおけるSticker / Collection仕様の正本とする。関連文書と矛盾する場合は本書を優先する。

## Sticker Board

- 検索バー、AppBar、下部ナビゲーションを除く領域を可能な限りボードとして使う。
- Stickerは自由配置、Collectionは整列展示とする。
- Freeは1ボード10枚、Premiumは1ボード30枚。旧上限20枚／100枚は廃止する。
- 元の高画質透過PNGを保持し、画面表示には最大640pxの透過プレビューを使用する。
- ボード共有には画面上のフチ、Shadow、テキスト、配置、回転、倍率を反映する。

## Stickerのモード

### 閲覧モード

- 許可: 1本指ドラッグによる移動。
- 禁止: 選択、回転、拡大縮小、削除、複製、貼り付け、デザイン編集。

### 編集モード

- タップでステッカーを選択する。
- 選択枠は移動、倍率、回転へ常に追従する。
- 選択枠上部に回転ハンドルを表示する。
- 回転中心はステッカー中心とし、ハンドル操作はピンチ拡縮と分離する。
- ピンチは拡大縮小のみを行う。
- ステッカー長押しでデザイン編集を開く。
- 貼り付け元を設定後、ボード長押し位置へ貼り付ける。

## 操作UIの役割

### 右上の≡メニュー

- 閲覧／編集モード切替。
- 選択中ステッカーの切り抜き再編集。
- 「貼り付けるステッカーを選択」は置かない。

### 選択時ツールバー

選択枠の下へ表示する。

- 貼り付け元に設定
- 複製
- 削除
- 拡大
- 縮小
- 最前面

最背面は実装しない。同一機能を≡メニューへ重複配置しない。

### 長押し

- ステッカー長押し: デザイン編集。
- 貼り付け元設定後のボード長押し: その位置へ貼り付け。

## Stickerデザイン

- 靴: 白い内フチ、太いオレンジ外フチ、任意のShadow。
- テキスト: オレンジ文字、白フチ、オレンジ外フチ。
- テキストはステッカーのオレンジフチ内へ重ねる。フチ外の独立ラベルにはしない。
- 靴詳細の「ステッカーテキスト」を作成時の初期値に使う。
- Sticker側で文字を変更しても靴詳細データは変更しない。
- 文字サイズとフチ内位置をステッカー単位で保存する。
- 文字サイズを考慮し、文字がフチ外へ出ない範囲へ移動を制限する。

## 切り抜き編集

- 自動切り抜き結果を確定後、消去／復元ブラシで微調整する。
- 既存ステッカーの再編集は保存済み透過PNGから開始する。
- オレンジ輪郭をタップすると、その輪郭で囲まれた誤透過領域全体を復元する。
- オレンジ輪郭は間引いて軽量表示する。
- ブラシ操作中は指と反対側へ拡大ルーペを表示する。
- 消去ブラシで閉じた線を描いた場合は内側を一括消去する。
- 復元ブラシでは囲み一括復元を行わない。

## Collection

- Collectionは2〜5列の整列表示とし、表示列数を保存する。
- 検索、ブランド／状態／カラー絞り込み、共有画像出力を提供する。
- 透過PNGを優先表示し、元写真は保持する。
- Sticker Boardの自由配置操作をCollectionへ持ち込まない。

## データと上限

- `sticker_path`: 再編集・高画質出力用の原本透過PNG。
- `preview_path`: 透明余白を切り詰めた、ボード表示用の最大640px透過PNG。
- `text_scale`, `text_x`, `text_y`: 文字サイズと正規化位置。
- 上限判定はFree 10枚／Premium 30枚を共通ポリシーとして扱う。
