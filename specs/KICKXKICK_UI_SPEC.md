# Kick×Kick UI Specification v1.1

## 1. ナビゲーション

Bottom Navigation:

- Home
- Collection
- ＋
- Sticker
- Settings

中央FAB:

- スニーカー追加

## 2. Home

最終構成:

1. TOP5
2. Collection
3. Sticker
4. 最近追加したスニーカー
5. Statistics

### TOP5

Home上部の約1/3を使う。

- 1位を大きく表示
- 2〜5位を小さく表示
- 👑アイコン
- Kick×Kickの顔になるエリア

## 3. Collection UI

Collection = 博物館 / 展示棚

### 起動時

最後に見ていた棚を表示する。
棚一覧からではなく、前回の棚を直接開く。

### 通常UI

- 左上: 棚一覧
- 中央: 棚名
- 右上: 編集

### 編集モード

下部ツールバー:

- スニーカー追加
- 背景変更
- 箱表示
- 並び替え
- 保存

### 棚一覧

サムネイル付き。

表示項目:

- 棚名
- 足数
- テーマ

### 倍率案内

初回だけ以下の案内を表示する。

> ピンチで表示倍率を変更できます

## 4. Collection倍率UI

ピンチで変更する。

- 2足表示
- 3足表示
- 4足表示
- 5足表示

前回倍率を記憶する。

倍率変更時は以下を同時に拡縮する。

- 靴
- 箱
- Display Title
- 名札

内部配置データは変わらない。

## 5. Collection配置UI

- 靴はスロットに吸着
- 自由配置ではない
- ドラッグで近いスロットへ吸着
- 保存後は配置固定

Collectionは整列展示。
自由配置はSticker担当。

## 6. 箱表示UI

箱写真がある場合のみ表示ON/OFFを選べる。

- 箱表示ON
- 箱表示OFF

デフォルトはOFF。

靴と箱を別々に探させない。
必ずスニーカーに紐づけて扱う。

## 7. Sticker UI

Sticker = 机 / スクラップブック

Stickerタブはステッカー素材一覧ではなく、乱雑に貼って遊ぶボード。

### Sticker Board

自由配置キャンバス。

できること:

- ドラッグ移動
- 長押し回転
- ピンチ拡大縮小
- 固定 / 解除
- 重なり順変更
- 最前面
- 複製
- 削除
- Undo
- Redo

## 8. ステッカー操作UI

### 短押し

固定 / 固定解除

### 長押し

指を動かさず自動回転

### ピンチ

拡大縮小

### 複製

同じステッカーを少しずらして配置

### Undo / Redo

30回

### 削除

- ゴミ箱へドラッグ
- 編集メニューから削除

## 9. ステッカー生成UI

生成フロー:

1. 写真
2. 自動背景削除
3. プレビュー
4. 必要なら修正
5. 保存
6. 透過PNG化

簡易修正:

- 消しゴム
- 復元
- ズーム
- Undo
- Redo

Photoshop的な編集はしない。

## 10. オンボーディング

初回のみ3画面。

### 1. Collect

スニーカーを登録

### 2. Create

ステッカー化

### 3. Exhibit

棚に展示

最後に「始める」。

- スキップ可能
- Settingsから再確認可能

## 11. 初回体験導線

オンボーディング
↓
最初の一足を登録
↓
ステッカー生成
↓
Sticker Boardへ

初回はHomeに戻さない。
Kick×Kickの価値であるステッカー体験まで進める。

## 12. デザインルール

カラー比率:

- 90% モノクロ
- 10% オレンジ

色:

- Black
- White
- Gray
- Orange

背景全面オレンジは使わない。
オレンジはアクセントとして使う。
