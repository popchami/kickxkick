# COLLECTION_SCREEN_SPEC.md

## 概要

Collection画面はユーザーが登録したスニーカーを一覧表示する画面である。

SoleMuseumにおけるギャラリーとして機能する。

---

# 目的

- コレクションを閲覧する
- スニーカーを検索する
- スニーカーを絞り込む
- 詳細画面へ遷移する

---

```text
│ □ □               │

```

# シューカード

## 表示項目

- 写真

- ブランド名
- 収蔵番号

```text

Nike

```

---

# 表示禁止項目

カード内に表示しない

- サイズ
- 購入店舗
- 購入価格
- メモ
