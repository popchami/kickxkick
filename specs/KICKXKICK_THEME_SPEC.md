# Kick×Kick Theme Specification

## Theme Lineup

### Free Theme
**Scrapbook Collage** (Default)
- Paper, masking tape, magazine cut-outs, handwritten notes.
- Monochrome base with orange accent.
- User sneaker photos remain the main focus.

### Premium Theme 1
**Pop Illustration Street**
- Black base with neon accent colors.
- Graffiti, stickers, speech bubbles, street-art motifs.
- Energetic, bold and playful.

### Premium Theme 2
**Pop Colorful Girly**
- Pastel color palette.
- Hearts, ribbons, stars and cute illustrations.
- Bright, cheerful scrapbook feeling aimed at users who enjoy cute designs.

## Theme Policy
- Free users can use the Scrapbook Collage theme.
- Premium users unlock all Premium themes.
- Themes change only the visual presentation and assets; app functionality remains identical.
