# Sticker Board 変更履歴仕様

本書は変更履歴であり、現行仕様の正本ではない。本文内容は `KICKXKICK_MVP_STICKER_COLLECTION_SPEC.md` へ反映済み。

## 2026-06-29

- 右上≡メニュー、選択時ツールバー、長押しの役割を分離。
- 閲覧モードでも移動のみ許可。
- 選択枠、回転ハンドル、ツールバーを配置・倍率・回転へ追従。
- 回転ハンドルをステッカー中心基準の角度計算へ変更。
- ピンチを拡大縮小専用へ変更。
- 長押しでデザイン編集を開く仕様へ変更。
- テキストをオレンジフチ内へ移動し、サイズ・位置編集を追加。
- Boardを利用可能領域いっぱいへ拡張。
- Board上限をFree 10枚、Premium 30枚へ変更。20枚／100枚を廃止。
- 高画質原本と最大640px表示プレビューを分離。
- 切り抜き輪郭タップ復元、輪郭軽量化、反対側ルーペ、囲み消去を追加。
