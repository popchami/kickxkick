# Free Experience Specification v1.0

## Purpose

Free users must understand the value of Kick×Kick before upgrading.

The free plan should not feel like a broken app.

It should feel like a complete small collection experience.

---

## Free Plan Limit

Free users can register up to 5 sneakers.

Reason:

- 5 sneakers is enough to experience the app
- 5 sneakers is small enough to create upgrade motivation
- 5 sneakers matches the idea of a starter collection

---

## Free User Experience Goal

After registering 5 sneakers, the user should feel:

```text
My collection looks good.
I want to add more.
```

The upgrade motivation should come from wanting to continue, not from frustration.

---

## Free Features

Free includes:

- 5 sneaker registrations
- Photo registration
- Search
- Favorites
- TOP5
- List display
- Shelf display
- Sticker display
- 1 sticker per sneaker
- Normal sticker style

---

## Free Sticker Rules

Free users can create one sticker per sneaker.

Allowed:

- Normal photo-based sticker
- White outline
- Soft shadow
- Display in Sticker Mode

Not included:

- Multiple stickers per sneaker
- Sticker editing
- PNG export
- LINE stamp material export
- Chibi / Cartoon / Pixel conversion

---

## Premium Upgrade Triggers

Upgrade prompts should appear when the user:

1. Tries to register the 6th sneaker
2. Tries to create a second sticker for the same sneaker
3. Tries to export a sticker PNG
4. Tries to use LINE stamp material export
5. Tries to use backup or restore
6. Tries to use sticker editing or conversion

---

## Upgrade Message Tone

Avoid negative wording.

Bad:

```text
This feature is locked.
```

Good:

```text
コレクションをもっと増やせます
Premiumで登録数を無制限にできます
```

Good:

```text
このステッカーを書き出すにはPremiumが必要です
```

---

## Premium Plan

Premium is a one-time purchase.

Price candidate:

```text
500円
```

Premium unlocks:

- Unlimited sneaker registration
- Backup
- Restore
- Data export
- Multiple stickers per sneaker
- Sticker editing
- PNG export
- LINE stamp material export
- Chibi conversion
- Cartoon conversion
- Pixel conversion

---

## Design Principle

Free should show the fun.

Premium should unlock continuation, safety, and output.

Free:

```text
Collect and enjoy 5 sneakers.
```

Premium:

```text
Build, protect, edit, and export your full collection.
```

---

## Definition of Done

This specification is complete when:

- Free plan has a clear 5-sneaker experience
- Premium has clear upgrade value
- Sticker limits are understandable
- Upgrade prompts feel positive, not punitive
